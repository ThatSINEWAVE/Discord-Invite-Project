The Discord Invite Project

Exploring the concept of a database to track and analyze Discord invite link.

The project aims to assist in Open Source Intelligence (OSINT) investigations by providing tools to identify and study Discord servers, including potentially malicious ones.

This repository serves as an educational and research platform exploring the concept of creating a database to track and analyze all possible Discord invite links.
